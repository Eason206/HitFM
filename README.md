# HitFM 电台存档

这是一个为纪念 **HitFM 电台** 而建立的存档仓库。  
从小到大，HitFM 伴随了我们无数个早晨与夜晚，它的音乐、节目和主持人的声音，是许多人青春记忆的一部分。无论是在上学的路上，还是放学回家的车上，它都是最忠实的陪伴。  

遗憾的是，自 **2025/12/22** 节目结束后，HitFM 就再也没有信号了。为了让这些美好的记忆得以延续，我建立了这个存档仓库，将自己收集到的音频文件整理保存。这里的内容全部为 **2025/12/23 之前**的节目，如果某些日期未更新，可能是暂时找不到对应资源，更新会尽量持续进行。  

希望这个仓库不仅是存档，也能成为大家回忆 HitFM 的一个小小窗口，让过去的旋律在记忆中继续流淌。

## 节目列表

- **Music Flow**：音乐流，全天候播放精选音乐  
- **Morning Call**：音乐叫早，温暖的早晨开始  
- **Hit Morning Show**：早间音乐，活力满满的节目  
- **At Work Network**：工作随身听，伴随你高效工作  
- **Lazy Afternoon**：慵懒下午茶，放松的午后音乐  
- **Big Drive Home**：开车现场秀，适合驾车时的热血伴奏  
- **New Music Express**：新音乐速递，及时推送最新音乐  
- **Hit FM Dance**：电音，带来劲爆节奏和舞曲体验  

每一档节目都承载着回忆，每一首歌都记录着我们生活中的点滴。愿这些存档，能让我们记住那些陪伴我们成长的旋律，延续 HitFM 带给我们的美好时光。
